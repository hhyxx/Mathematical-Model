
% matlab 求解线性规划的命令
[x,fval] = linprog[c,A,b,Aeq,beq,lb,ub,x0] 
% c 是 目标函数的系数 
% A 不等式约束的系数
% b 不等式约束的函数值 
% Aeq 等式约束的系数
% beq 等式约束的函数值
% lb 下界
% ub 上界
% x0 表示初始值

% 若不存在不等式约束，就使用 [] 代替A和b 
% 若不存在等式约束，就使用 [] 代替Aeq 和beq
% 无上界 lb(i)=inf 无下界 ub(i) = -inf
% 返回值x表示最小值处的x取值，fval表示最优解 函数的最小值
% 无解 和无穷解
% 求最大值则给fval 添加一个负号
