# Mathematical-Model

## 1 建模分析
模型的建立
+ 描述或解释现实世界
    + 机理分析法研究事物的内在规律
+ 预测、预报
    + 概率统计、优化理论、模拟计算
+ 优化管理、决策、控制
    + 引入量化的评价指标和评价方法

解的分析与检验
+ 对解的实际意义进行分析
    + 即模型的解在实际意义中说明了什么、效果怎样、模型适用范围如何
    + 误差分析和灵敏度分析
+ 误差产生原因
    + 模型假设引起的误差
    + 近似求解方法引起的误差
    + 计算机舍入问题产生的误差
    + 数据本身的误差
+ 灵敏度分析
    +  给参数一个很小的扰动，看结果的扰动是否也很小
